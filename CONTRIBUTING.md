# Contributing

Thanks for considering contributing to AI Workspace Main Site.

## Quick Start

```bash
git clone https://github.com/zhiwoosa-lab/main-site.git
cd main-site
open index.html
```

No build step. Edit any HTML/CSS/JS file and refresh your browser.

## How to Contribute

- **Content** — Add journal entries in `content/{section}/journal/` as Markdown with YAML frontmatter
- **Design** — Edit `styles.css` for visual system changes
- **Docs** — Improve or fill in files under `docs/`
- **Skills** — Add OpenClaw skill definitions under `skills/`

## Guidelines

- Keep it simple — no build tools, no frameworks
- Maintain the minimal, editorial visual style
- Document your changes in the relevant `docs/` file
- Run `open index.html` to preview before submitting

## Submitting Changes

1. Fork the repo
2. Create a feature branch
3. Commit your changes
4. Submit a pull request

## Questions?

Open an issue or reach out via the project's discussion channel.
