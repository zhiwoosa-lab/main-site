# Project

## Goal

为 AI Workspace 创建一个高端、克制、可直接访问的主站首页，清晰传达其面向长期 AI 协作开发的定位。

## Current Status

- 已初始化静态网站骨架
- 已建立首页信息架构与基础交互
- 已引入首屏视觉素材

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## UI Style

- Minimal
- Elegant
- High-end
- Editorial but practical
- Desktop / mobile responsive

## Deployment Method

当前为静态页面，可直接部署到：

- GitHub Pages
- Cloudflare Pages
- 任意静态托管服务

## Constraints

- 保持页面加载轻量
- 不依赖构建工具
- 不修改 OpenClaw 本体
- 后续新增项目内容时保持结构清晰

## Future Roadmap

- 增加真实项目案例页
- 增加自动化与 Agent 能力详情页
- 迁移到 Next.js
- 接入表单或联系入口
