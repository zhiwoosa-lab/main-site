# Typography

> Editorial, modern, readable at any size.

## Font Stack

```
-apple-system, BlinkMacSystemFont, "SF Pro Text",
"Helvetica Neue", Arial, sans-serif
```

For CJK content, falls back to system Chinese fonts.

## Scale

| Level | Size | Weight | Usage |
|---|---|---|---|
| Display | 48px | 700 | Hero headings |
| Title | 28px | 600 | Section titles |
| Heading | 20px | 600 | Card headers |
| Body | 16px | 400 | Main content |
| Small | 14px | 400 | Meta, captions |
| Micro | 12px | 500 | Labels, badges |

## Line Height

- Headings: 1.2
- Body: 1.6
- Small: 1.4

## CJK Notes

Chinese text uses same scale but may need tighter line-height (1.4).
