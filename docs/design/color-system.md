# Color System

> The visual palette for AI Workspace Main Site.

## Core Palette

| Token | Hex | Usage |
|---|---|---|
| `--bg-primary` | `#0a0a0f` | Page background |
| `--bg-card` | `#111118` | Card/section backgrounds |
| `--text-primary` | `#e8e8ef` | Body text |
| `--text-secondary` | `#8888a0` | Muted / meta text |
| `--accent` | `#6c8cff` | Links, highlights, interactive |
| `--border` | `#1e1e2a` | Subtle dividers |

## Semantic Colors

| Token | Hex | Usage |
|---|---|---|
| `--success` | `#4ade80` | Positive metrics |
| `--warning` | `#fbbf24` | Attention, caution |
| `--error` | `#f87171` | Issues |
| `--info` | `#38bdf8` | Information |

## Apple Colors

- **Red Apple** — `#ef4444` (achievements)
- **Green Apple** — `#22c55e` (lessons)
- **Gift** — `#f59e0b` (life-changing moments)
- **Rotten Apple** — `#6b7280` (loss)

## Usage

- All colors maintain contrast ratio above 4.5:1 for body text
- Accent colors used sparingly for emphasis
