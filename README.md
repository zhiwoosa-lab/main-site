# AI Workspace Main Site

> A minimal, editorial personal website built with OpenClaw — showcasing life, health tracking, language learning, and AI-assisted content workflows.

This is the main site for **AI Workspace**, an OpenClaw-powered personal automation environment. The site serves as a living dashboard for ongoing life stages, health metrics, language learning progress, and AI lab experiments.

## Features

- **Life Tree** — Interactive timeline of current life stage (Johannesburg, South Africa)
- **Chronicle** — Personal event logging
- **Health Dashboard** — Real data from Apple Health (weight, body fat, sleep, heart rate, activity)
- **Language Log** — Language learning progress tracking
- **Skills Library** — AI skill definitions used in OpenClaw workflows
- **AI Lab** — Personal AI projects and experiments

## Tech Stack

- Plain HTML, CSS (vanilla), minimal JavaScript
- Designed with editorial typography and responsive layout
- No build tools required — open `index.html` in any browser
- Deployable to GitHub Pages, Cloudflare Pages, Vercel, or any static host

## Project Structure

```
├── index.html           # Main entry — Current Stage Tree
├── styles.css           # Visual system & responsive layout
├── script.js            # Lightweight interactions
├── chronicle/           # Life event archive
├── health/              # Health metrics dashboard
├── language/            # Language learning log
├── skills/              # OpenClaw skill definitions
├── content/             # Structured content (health journal, metadata)
├── docs/                # Design system, content model, workflow docs
├── assets/              # Hero images & visual assets
├── apple/               # Apple ecosystem reference
└── south-africa/        # South Africa living guide
```

## Getting Started

```bash
# Clone the repo
git clone https://github.com/zhiwoosa-lab/main-site.git

# Open in browser
open index.html
```

No build step. No dependencies. Edit and refresh.
